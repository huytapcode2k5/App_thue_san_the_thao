# App_thue_san_the_thao
I.Giới thiệu hệ thống

Ứng dụng “Thuê sân thể thao” là ứng dụng di động được phát triển nhằm hỗ trợ người dùng tìm kiếm, xem thông tin và đặt sân thể thao trực tiếp trên điện thoại.
Ứng dụng hỗ trợ các chức năng cơ bản như:
-Đăng ký, đăng nhập tài khoản
-Xem danh sách sân thể thao
-Xem chi tiết sân
-Đặt sân theo ngày và giờ
-Xem lịch sử đặt sân
-Thanh toán
-Quản lý thông tin người dùng
Dự án được thực hiện trong quá trình học tập môn “Lập trình trên thiết bị di động”.

V. Hướng dẫn cài đặt
1. Clone project
git clone https://github.com/ten-github/App_thue_san_the_thao.git
2. Di chuyển vào thư mục project
cd App_thue_san_the_thao
3. Cài đặt thư viện
npm install
+)Hướng dẫn chạy project
Chạy ứng dụng bằng Expo
npx expo start
Sau khi chạy lệnh trên:
Nhấn phím a để chạy Android Emulator
Hoặc quét QR bằng ứng dụng Expo Go trên điện thoại
Ngoài ra có thể dowload app bằng link apk : https://expo.dev/accounts/nvanhuy2k5/projects/ThueSanTheThaoApp/builds/48c49e38-f685-4145-91b0-86778628f127
